Welcome to the **Seafoam Labs Wiki** – your starting point for everything we build.

Seafoam Labs creates open-source tools for Arch Linux and beyond. Browse the sections below to find documentation, guides, and resources for each project.

---

## Projects

### 🐚 Shelly ALPM

A modern, visual package manager for Arch Linux built with C# and GTK4. Shelly ALPM is a Pacman alternative designed with usability and aesthetics in mind, with support for CachyOS, Garuda Linux, and more.

- [Overview](/shelly-alpm/overview) – What Shelly is and why you'd use it.

---

### ⭐ Starfish

A dependency viewer for Arch packages. Starfish helps you inspect and understand package relationships visually, making it easier to explore dependency trees and connections.

- [Overview](/starfish/overview) – Features and how to get started.

---

### 💧 Aqueous

A minimal Wayland window manager built in C# and .NET 10 on top of RiverDelta, Seafoam Labs' in-tree fork of [River](https://codeberg.org/river/river). Aqueous provides a lightweight, configurable Wayland environment with compatible shell integrations, configurable layouts, and Arch-ready packaging.

- [Overview](/aqueous/overview) – Architecture, features, and the repository.

---

### 🪸 Pori

A desktop tool for creating reliable, persistent Linux mounts using native systemd `.mount` units. Pori helps you configure local disk mounts without manually editing `/etc/fstab`.

- [Overview](/pori/overview) – What Pori does and current scope.
