## Installation

The recommended installation for Shelly is for CachyOS or using CachyOS packages:

<Tabs>
  <TabItem label="pacman">
    <Code code="sudo pacman -S shelly" lang="sh" />
  </TabItem>
  <TabItem label="yay">
    <Code code="yay -S shelly" lang="sh" />
  </TabItem>
  <TabItem label="paru">
    <Code code="paru -S shelly" lang="sh" />
  </TabItem>
</Tabs>

## Why Shelly?

<CardGrid stagger>
  <Card title="Native Performance" icon="rocket">
    Directly interacts with libalpm for the fastest and most accurate package
    management possible.
  </Card>
  <Card title="Modern UI" icon="laptop">
    Built with GTK 4, providing a beautiful, responsive, and clean user
    interface.
  </Card>
  <Card title="AUR & Flatpak" icon="puzzle">
    Integrated support for Arch User Repository and Flatpaks, managing all your
    software in one place.
  </Card>
</CardGrid>
