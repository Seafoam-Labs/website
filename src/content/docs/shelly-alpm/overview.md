# What is Shelly?

Shelly is a fresh, modern way to manage software on Arch Linux and Arch-based systems. It’s designed to be simple, friendly, and visually appealing - making it easy for anyone to install, update, or remove apps and packages, whether you prefer a beautiful desktop app or a powerful command line.

---

## Why Use Shelly?

- **Easy to Use:** Clean, intuitive interface - no need to memorize commands.
- **All-in-One:** Search and manage software from official Arch repositories, the AUR, Flatpak and even AppImages, all in one place.
- **Modern Look:** Built with the latest technology for a smooth, native experience.
- **Flexible:** Use the desktop app or the command line, whichever you prefer.

---

## What Can You Do With Shelly?

- **Find Apps Fast:** Search for software from official sources, the AUR, or Flatpak.
- **Install & Remove:** Add or remove apps with just a click or a simple command.
- **Stay Updated:** See what needs updating and upgrade everything easily.
- **Customize:** Choose between a graphical interface or command line - whatever fits your style.

---

## Frequently Asked Questions

**Do I need to be a Linux expert to use Shelly?**  
No! Shelly is designed for everyone, from beginners to power users.

**Can I use Shelly on any Arch-based distro?**  
Yes, Shelly works on Arch Linux and most Arch-based distributions.

**Is Shelly free?**  
Absolutely! Shelly is open source and always will be.
