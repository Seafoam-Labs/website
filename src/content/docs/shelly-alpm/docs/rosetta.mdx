Shelly Rosetta is a reference for pacman commands and their equivalent Shelly commands. It is intended to help users who are familiar with pacman to transition to Shelly.

## Install or update

| pacman                | shelly (short/long)                                            | Description                                                   |
| --------------------- | -------------------------------------------------------------- | ------------------------------------------------------------- |
| `pacman -S package`   | `shelly -SI package` <br/> `shelly install package`            | (Re)Install a package                                         |
| `pacman -Syu`         | `shelly -SU` or `shelly` <br/> `shelly upgrade`                | Perform a system update                                       |
| `pacman -Syu package` | `shelly -SIu package` <br/> `shelly install --upgrade package` | Install a package and do a system update                      |
| `n/a`                 | `shelly -UK` <br/> `shelly check-updates`                      | Check for updates                                             |
| `pacman -U URL`       | `shelly -SI URL` <br/> `shelly install URL`                    | Download and install a .pkg.tar.zst package from the internet |

## Search or information

| pacman/related       | shelly (short/long)                                                      | Description                                                                                                                                              |
| -------------------- | ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `pacman -Ss package` | `shelly -SQa package` <br/>`shelly query --available package`            | Search the arch/cachy repos                                                                                                                              |
| `pacman -Si package` | `shelly -SQad package` <br/> `shelly query --available --detail package` | Get detailed information about a package in the arch/cachy repos                                                                                         |
| `pacman -Qs package` | `shelly -SQi package` <br/> `shelly query --installed package`           | Search locally installed package names                                                                                                                   |
| `pacman -Qi package` | `shelly -SQ package` <br/> `shelly query --installed --detail package`   | Get detailed information for a locally installed package                                                                                                 |
| `yay -Ss package`    | `shelly -ASs package` <br/>`shelly aur search --standard package`        | Search for a package in the standard repos and AUR `alias dat-ass="shelly -ASs"`                                                                         |
| `yay -Si package`    | `shelly -ASd package` <br/>`shelly aur search --detail package`          | Get detailed information about a package in the AUR                                                                                                      |
| `pacman -Qm`         | `shelly -AL` <br/> `shelly aur list`                                     | List packages installed from the AUR.                                                                                                                    |
| `pacman -Qe package` | `n/a`                                                                    | List all explicitly installed packages. This is primarily used for package list backup purposes like with `pacman -Qqe`, but `shelly export` covers that |

## Remove

| pacman                          | shelly (short/long)                                            | Description                                                                                                                                                                   |
| ------------------------------- | -------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `pacman -R package`             | `shelly -SR package` <br/>`shelly remove package`              | Remove a single package, leaving all of its dependencies installed                                                                                                            |
| `pacman -Rs package`            | `shelly -SR package` <br/>`shelly remove --cascade package`    | Remove a package and its dependencies which are not required by any other installed package                                                                                   |
| `pacman -Rsc package`           | `shelly -SRi package` <br/> `shelly remove --cascade --ripple` | Remove a package, its dependencies and all the packages that depend on the target package. <Badge text="Be very careful running this" size="small" variant="danger" />        |
| `pacman -Rdd package`           | `shelly -SRf` <br/> `shelly remove --force`                    | Remove a package, skipping any dependency checks. <Badge text="This can break your system. Only run if advanced user or directed by support" size="small" variant="danger" /> |
| `pacman -Sc`                    | `shelly -UCi` <br/>`shelly cache-clean --uninstalled`          | Clean the local package cache of uninstalled packages                                                                                                                         |
| `pacman -Scc`                   | `shelly -UCk 0` <br/> `shelly cache-clean --keep 0`            | Clean the local package cache of all packages                                                                                                                                 |
| `pacman -Qdtq \| pacman -Rns -` | `shelly -SPo` <br/> `shelly purify --orphans`                  | Remove packages that were installed as a dependency and are no longer required by any package, aka orphans                                                                    |

## Downgrading

| pacman/related      | shelly (short/long)                                   | Description                                                                                          |
| ------------------- | ----------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| `downgrade package` | `shelly -SD package` <br/> `shelly downgrade package` | Revert a package you have installed to an earlier cached version or cachy repo version if available. |
