Welcome to the Seafoam Labs blog! We're a small team building open-source tools for Arch Linux and beyond.

## What We're Working On

Our current projects include:

- **[Shelly ALPM](/shelly-alpm)** — a modern, visual package manager for Arch Linux built with C# and GTK4.
- **[Starfish](/starfish)** — a dependency viewer for Arch packages.
- **[Aqueous](/aqueous)** — a new project currently in early development.

## What's Next

We'll be using this blog to share updates, release notes, and the occasional behind-the-scenes look at how we build things. Stay tuned!
