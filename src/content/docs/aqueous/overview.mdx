**Aqueous** is a minimal **Wayland window manager** built on top of a customied [River](https://codeberg.org/river/river) for called RiverDelta.

It is designed as a lightweight, configurable River-based environment with a separate shell/bar layer powered by the external [Noctalia](https://github.com/noctalia-dev/noctalia-shell) project.

- **Language:** C# (primary), zig (secondary), with some shell.
- **Platform focus:** Linux / Wayland

## Features

Aqueous provides a tiling oriented window management stack with:

- a core window manager client
- an output daemon for display/output control
- test projects for validation
- packaging and session-launch integration for Linux environments

The project also includes configuration, packaging, and session assets to support a full desktop login flow.
